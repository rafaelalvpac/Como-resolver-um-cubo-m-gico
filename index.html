Como resolver um cubo mágico 3x3x3

Resolver um cubo mágico 3x3x3 pode parecer desafiador, mas com prática e paciência, você pode aprender a resolvê-lo. Aqui nesta página, vou te mostrar os passos para montar o cubo mágico.

Metodo Básico

O método básico envolve a resolução do cubo em camadas, são usados 7 passos. É necessario saber as letras que representam os movimentos,Clique Aqui ,para conhece-los.

Passo 1: Preparar a cruz branca.

Passo 2: Finalizar a cruz branca.

Passo 3: completar a camada branca posicionando as peças dos cantos.

Passo 4: Resolver a segunda camada com as peças do duas cores, chamadas de meios.

Passo 5: Fazer a cruz na face amarela.

Passo 6: resolver a face amarela

Passo 7: Posicionar as peças de quina da face amarela.

Passo 8:Posicionar os meios na ultima camada, para finalizar o cubo.

Segue abaixo os como realizar passo a passo cada etapa acima

Passo 1: Preparação da cruz:

Este é um dos casos mais faceis de resolver onde há, uma peça branca na segunda camada e a posição desejada no topo na face da direita. Fórmula:R.

Este é outro caso de cruz onde a peça branca está na face, mas na primeira camada e a posição desejada no topo. Fórmula:F' R.

Este é um caso de cruz onde a peça branca esta na segunda camada, mas a posição desejada está ocupada por outra peça. Fórmula:U' R.

Este o caso onde a peça branca esta na face na segunda camada, e a posicão desejada, no topo. Fórmula:F U' R.

Passo 2: Finalização da cruz:

o objetivo deste passo é coclocar as peças posicionadas no topo alinhadas com as peças de centro, e depois colocá-las na primeira camada com o movimento de F2.

Depois disso, faça o movimento F2 para colocá-la na primeira camada.

Assim a cruz branca está finalizada, então pode-se seguir para o terceiro passo do tutorial.

Passo 3: Primeira camada:

No terceiro passo o objetivo é completar a camada branca. É preciso encontrar uma peça de quina que tenha um lado branco que esteja para o lado direito ou esquerdo e fazer o movimento: R U R' U'.

Neste caso é necessário repetir o movimento: R U R' U' 5 vezes para encaixar a peça no lugar certo.

Neste outro exemplo, a quina precisa estar na primeira camada com o lado certo. Para resolve-la é necessário fazer o movimento: R U2 R' U' R U R'.

Seguindo todos estes passos, o terceiro passo está concluido, pode-se seguir para o quarto passo do tutorial.

Passo 4: Segunda camada

No quarto passo o objetivo é completar a segunda camada do cubo mágico. É necessário encontrar uma peça de meio que esteja na camada de cima que não tenha a cor amarela, posicionar a cor debaixo da peça junto com a cor do centro de mesma cor, e executar o movimento: Fórmula: U R U' R' U' F' U F.

Neste caso é necessário repitir os mesmos movimento mas espelhado (do outro lado do cubo) para encaixar a peça de meio na segunda camada. Fórmula: U' L' U L U F U' F'.

Neste caso há um a peça no lugar errado, para consertar essa situação basta fazer o mesmo movimento anterior do lado direito e faze-lo novamente no lugar certo para encaixar a peça de meio na segunda camada se for o primeiro caso, faça o movimento dele, se for o segundo, faça o movimento dele.

Assim a segunda camada está completa se feito esses movimentos com as 4 peças de meio da segunda camada, pode-se seguir para o quinto passo do tutorial.

Passo 5: Cruz amarela

No quinto passo do cubo mágico o objetivo é fazer a cruz amarela. Para isso, é necessário fazer o movimento F R U R' U' F' para formar a cruz amarela.

No quinto passo do cubo mágico o objetivo é fazer a cruz amarela. Neste caso, chamado de caso L a posição dele para a execução do movimento F R U R' U' F' é de um "l" em pé e invertido.

Neste caso, chamado de caso linha, a posição dele para a execução do movimento F R U R' U' F' é de uma linha horizontal.

Assim a cruz amarela está completa, se foram realizados os movimentos ensinados, pode-se seguir para o sexto passo do tutorial.

Passo 6: Face amarela

Este é o sexto passo do tutorial, onde o objetivo é resolver a face amarela, sera usado o mesmo movimento e a face amarela irá se transformar em algum dos casos do sexto passo. Neste caso é necessário fazer o movimento R U R'U R U2 R'.

Neste caso o movimento é o mesmo, a posição dele é baseada nas peçã viradas para o lado: uma virada para voçê e a outra para frente, como mostra a imagem.

Neste caso, o movimento é o mesmo, mas a posição das duas peças é diferente, como se os dois casos fossem gêmeos, a posição das duas peças sera elas viradas para voçê.

Neste caso, o movimento é o mesmo, mas o desenho formado de de uma cruz com duas peças viradas para o lado direito e outras duas para o lado esquerdo.

Este caso é um"gêmeo" do ultimo caso apresentado mas as duas peças viradas da direita estão uma para a frente e a outra para trás, a posição para a execução do movimento é as duas viradas da esquerda para a esquerda e as duas viradas da direita na direita.

Este caso é conhecido como caso do peixe, tendo duas variações, este a cabeça do peixe precisa estar posicionada na diagonal para a esquerda, para ser executado o movimento.

Este caso é a outra variação do caso do peixe, pois ha uma peça amarela ao lado da cabeça do peixe, a posição para execução do movimento é a mesma.

se voçê seguiu todos os ensinamentos do sexto passo seu cubo deve estar com ele finalizado, se sim siga para o sétimo passo.

Passo 7:Finalizar as quinas

O passo 7 tem como objetivo colocar no lugar certo cada quina da camada do topo. Neste caso há duas peças de mesma cor e o meio esta com outra cor. Será necessário colocar as duas peças de mesma cor para baixo e a face amarela para sua frente e executar o movimento: R U' R D2 R' U R D2 R2.

Se não houver nenhum par de quinas no topo, faça o movimento do ultimo caso apresentado e volte a procurar por eles, se contunuar veja se há algo de errado.

Assim é finalizado o passo 7 executando os movimentos certos, siga para o passo 8 para continuar.

Passo 8: Finalizar os meios

No passo 8 o objetivo é finalizar o cubo por completo colocando os meios no lugar certo. Neste caso você precisa verificar a ordem que as peças precisam trocar de lugar, veja para onde cada uma vai e veja se está no sentido horário ou anti-horário pois no sentido anti-horário será necessário aplicar a fórmula F2 U R'L F2 R L' U F2, e deixe a face que está resolvida para trás.

Este é o caso anti-horário, ele é necessário repetir a fórmula 2 vezes deixando a face resolvida do mesmo jeito.

Se não houver nenhum destes casos repita uma das fórmulas anteriores e veja se há um dos casos de novo.

Parabéns, você acaba de resolver o cubo magico 3x3x3! pratique as fórmulas e melhore seues tempos para poder resolver rápido.
